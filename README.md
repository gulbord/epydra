# ESEDRA File Cleaning Tool

Install [uv](https://docs.astral.sh/uv/getting-started/installation/) first (recommended).

```console
git clone https://github.com/gulbord/epydra
cd epydra
uv tool install -e .
```

Then the executable is globally available as `epydra`. Get help with `epydra`, `epydra -h` or `epydra --help`.
